# style-scraper
Scrape styles from a website [fonts and colors]

## Setup

### Pre-requisites

- OS: Ubuntu

- nodejs
  - node version 18

```
curl https://raw.githubusercontent.com/creationix/nvm/master/install.sh | bash 
source ~/.bashrc
source ~/.profile

nvm install 18
nvm use 18
```

```
./install.sh
```

```
npm test
```
